# Contributing

## Overview

All projects under **authn-cn** are currently maintained by a single contributor.
External contributions are not accepted at this time — please file an issue for any feedback, bug reports, or feature ideas instead.

## Projects

| Project | Description |
|---------|-------------|
| [authn-mock](https://github.com/authn-cn/authn-mock) | Mock OIDC OP & SAML IdP (Cloudflare Workers), test/dev only |
| [docs-site](https://github.com/authn-cn/authn-cn.github.io) | Documentation site (VuePress, multi-language: zh/en/de) |

## Filing Issues

Please use the appropriate issue template when creating a new issue. Each project may have its own specialized templates; if none exist for your component, a general-purpose template is also available.
